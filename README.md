# 酷云微信营销系统（KyPHP）

#### 介绍
基于thinkphp5.1的微信营销系统

#### 软件架构
- thinkPHP5.1
- EasyWeChat
- Mysql
- Memcached & Redis


#### 安装教程

1.  拉取项目
2.  将项目根目录下的env文件改为.env
3.  修改runtime 和 public/uploads的读写权限为777
4. composer update
